# TLMK Studio — Photography Website

## About
TLMK Studio is a modern cinematic photography studio based in London, UK.
Specialising in modern portraits, editorial, and urban street photography.
Instagram: @tlmkstudio · Bookings via DM.

The website's job: let the photography be the hero. Dark, minimal, editorial.
The images do all the talking — the design stays out of the way.

## Tech stack
- Plain HTML, CSS, and vanilla JavaScript. No framework.
- No build step. Files open directly in a browser.
- Mobile-first and fully responsive.
- Optimised, lazy-loaded images.

## File structure
- index.html       → Home (hero, about, gallery preview, contact)
- portfolio.html   → Full gallery grid
- about.html       → About the studio
- contact.html     → Booking form
- /css/style.css   → All styles
- /js/main.js      → Interactions (nav, lightbox, etc.)
- /images/         → All photography

## Design system
- Background:      #0A0A0A (near black)
- Text primary:    #FFFFFF
- Text muted:      rgba(255,255,255,0.45)
- Text faint:      rgba(255,255,255,0.25)
- Borders:         rgba(255,255,255,0.1)
- NO accent colour. The photos provide all the colour.

## Typography
- All-caps logo and labels with wide letter-spacing (0.2em–0.3em).
- Large, light-weight display headings (font-weight 400).
- Small uppercase eyebrow labels above sections.
- Body text small, muted, generous line-height (1.8).

## Layout rules
- Generous whitespace. Lots of black breathing room.
- Asymmetric image grids (not all equal squares).
- Full-bleed hero image.
- Sections separated by faint 0.5px white borders.
- Gallery items numbered TLMK / 001 style.

## Tone of voice
Confident, minimal, no filler. "Book a session" not "Click here to enquire now!".
Sentence case in copy. Never salesy.

## Working preferences
- When I ask for a change, edit the real files — don't paste code in chat.
- Keep the CSS organised by section with comments.
- Always check it still works on mobile.
