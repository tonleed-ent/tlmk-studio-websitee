# TLMK Studio Agent Guide

This is the existing live TLMK Studio website project. The site, custom domain, hosting, WhatsApp Business, Instagram, Meta ads, Git repository, and production assets already exist and must be treated as live business surfaces.

Use the current project structure and the workspace skills:

- [TLMK Studio Developer](.claude/skills/tlmk-studio-developer/SKILL.md)
- [TLMK Studio Styling](.claude/skills/tlmk-styling/SKILL.md)

Preserve the current design, hosting, domain, WhatsApp, Instagram, Meta ads, prices, packages, policies, contact details, images, videos, and production assets unless the owner explicitly approves a specific change.

Prioritise enquiries, paid bookings, mobile usability, performance, accessibility, and simple website management.

Work efficiently: read only targeted files needed for the request, reuse inspected context, prefer existing HTML/CSS/JavaScript patterns, and avoid broad repository scans unless the task requires them.

Get explicit approval before installing dependencies, adding services, introducing a backend, CMS, or authentication, changing hosting or DNS, creating API keys, committing, pushing, or deploying.
